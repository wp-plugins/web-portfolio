== Description ==

Web portfolio plugin is highly suitable to showcase your portfolio and group them nicely under jQuery powered tabs.
Features
* Admin interface to add, edit and manage portfolio
* Fields for link and Technologies Used details
* Filters to manipulate portfolio output

Shortcodes

[web_portfolio] - For listings on page or posts

Theme Functions

web_portfolio_list() - For listings on page or posts

Support

So that others can share in the answer, please submit your support requests through the WordPress forums for Web Portfolio(http://wordpress.org/support/plugin/web_portfolio).

== Installation ==

1.     Via WordPress Admin > Plugins > Add New, Upload the web_portfolio.zip file
1.     Alternately, via FTP, upload web_portfolio directory to the /wp-content/plugins/ directory
1.     Activate the 'Web Portfolio' plugin after uploading or through WordPress Admin > Plugins

Usage

1. Add and manage the quotes through the 'Web Portfolio' menu in the WordPress admin area
1. Configure the 'Web Portfolios' to select quotes and display as needed
1. Use the [web_portfolio] shortcodes to display testimonials on a page or in a post
1. Read FAQ 1 for web_portfolio_list() theme functions usage
