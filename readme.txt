== Description ==

Web portfolio is free, responsive and colorfull wordpress plugin can transform a humble wordpress website into a feature rich media gallery where you can proudly showcase your projects, client logos, photography, or any other pictures or videos of your choice. If you read through our documentation we have provided step by step instructions to use the various features and functionality of this plugin

Features
* Admin interface to add, edit and manage portfolio
* Fields for link and Technologies Used details
* Filters to manipulate portfolio output

Shortcodes

[web_portfolio] - For portfolio listings on page or posts

Theme Functions

web_portfolio_list() - For portfolio listings on page or posts

== Installation ==

1.     Via WordPress Admin > Plugins > Add New, Upload the web_portfolio.zip file
2.     Alternately, via FTP, upload web_portfolio directory to the /wp-content/plugins/ directory
3.     Activate the 'Web Portfolio' plugin after uploading or through WordPress Admin > Plugins

Usage

1. Add and manage the portfolio through the 'Web Portfolio' menu in the WordPress admin area
2. Configure the 'Web Portfolios' to select portfolio and display as needed
3. Use the [web_portfolio] shortcodes to display portfolio on a page or in a post
4. Read FAQ 1 for web_portfolio_list() theme functions usage
